POO - Definições:
- Classe(Concreta ou abstrata)
    É a junção de atributos e métodos que geram uma classe
    Ex: Usuário
- Objeto
    Obtem caracteristicas próprias e representam  as coisas do mundo real, então as diferença entre as classes é definida pelo objeto que é como uma instância da classe
    Ex: Professor
- Instância
    A classe gera o objeto a partir da instancia que tem suas caracteristicas únicas de uma determinada classe
    Ex: Se o professor é um moderador ou não
- Atributos
    São as caracteristicas que podem ser tanto do objeto quanto da classe, diferencia os tipos de classe entre elas e os tipos de objetos
    Ex: Nome e etc..
- Métodos
    São formas como o objeto se comporta dentro de uma determinada classe
    Ex: Professor faz empréstimo
- Constante
    É um valor único que não vai se alterar, portanto pra reduzir o trabalho ele é armazenado de forma que pode ser buscado sempre
    Ex: ID
- Interface
    Classe se compromete a fornecer a lógica para os métodos definidos, garantindo que diferentes partes do sistema possam se comunicar de maneira uniforme. 
- Modificadores de acesso
    Eles ajudam a organizar e proteger dados, evitando que informações sensíveis sejam alteradas de forma indevida.
